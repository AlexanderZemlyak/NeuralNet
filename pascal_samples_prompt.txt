Создай мне датасет для обучения deepseek-coder-1.3b-instruct из прикрепленных файлов с кодом на PascalABC.NET.
Датасет должен быть в виде json, у примеров должны быть ровно два строковых атрибута: "instruction" и "output".
Комментарии на русском в файлах переводи на английский, строковые значения переводить не надо. 
Если файлы очень длинные, то можешь разделять их на несколько отдельных примеров, опускать код из таких файлов не надо.
Если файлы похожие, то можешь из них составлять один пример.
При генерации создавай один json массив с учебными примерами внутри.
